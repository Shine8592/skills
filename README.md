# Sundial Skills

The [`skill`](https://github.com/sundial-org/skills/tree/main/skills/skill) skill can help you find, improve, and publish skills — install it with `npx sundial-hub add skill`.

You'll be able to peruse this library of curated skills, plus 50,000+ more aggregated from the web. Browse the [`skills/`](https://github.com/sundial-org/skills/tree/main/skills) directory or install any skill with:

```sh
npx sundial-hub add <skill-name>
```
