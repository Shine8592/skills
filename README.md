# Sundial Skills

A library of curated skills for Claude Code. Browse the [`skills/`](https://github.com/sundial-org/skills/tree/main/skills) directory or install any skill with:

```sh
npx sundial-hub add <skill-name>
```

The [`skill`](https://github.com/sundial-org/skills/tree/main/skills/skill) skill can help you find, improve, and publish skills — install it with `npx sundial-hub add skill`.

Created by [Sundial](https://sundialhub.com/)
