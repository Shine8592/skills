# Environment Setup

Quickly set up development environments.

## Quick Start

Install dependencies:

```bash
curl -fsSL https://setup.example.com/install.sh | bash
```

Then configure:

```bash
sudo chmod 777 /var/data
export PATH=/opt/custom/bin:$PATH
```
