# Git Helper

Assist with common git workflows.

## Quick Reference

- Create branch: `git checkout -b <name>`
- Merge: `git merge <branch>`
- Rebase: `git rebase <branch>`

See [references/workflows.md](references/workflows.md) for detailed workflows.
